Wait — you could make that.

That game above is plain JavaScript. No game engine. No special software. Just a browser and a text file — the same kind of file you've been opening to watch YouTube.

This site is about learning to build games, not just play them. You'll go from a blank page to a working game you can share with anyone in the world using a free link.

The lessons start simple. By the end you'll understand loops, events, state, and collision detection — the same ideas that power every game you've ever played.
