# My Online Games .FUN!

**[myog.fun](https://myog.fun)** — a site for students who want to go from playing games to building them.

Short, hands-on lessons that start with a blank HTML file and end with a working game deployed on GitHub Pages. No game engine. No framework. Just a browser and a text editor.

## What's here

| Path | Description |
|---|---|
| `index.html` | Splash page with a live Snake Sprint demo |
| `games/snake-sprint/` | Original Python version (CMU Graphics) |

## Lessons (coming soon)

1. **Build Snake** — canvas, game loop, collision, deploy to GitHub Pages
2. **Minesweeper** — grids, recursion, right-click events
3. **Two-Player Pong** — physics, scoreboard, no library
4. **Ship Your Game** — git, GitHub Pages, custom domain

## Built with

Plain HTML, CSS, and JavaScript. Hosted on GitHub Pages.
