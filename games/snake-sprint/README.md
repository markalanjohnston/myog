# Snake Sprint — CMU Graphics (Python)

The original Snake Sprint game built with the [CMU CS Academy](https://academy.cs.cmu.edu/) graphics library.

## How to run

1. Open the [CMU CS Academy IDE](https://academy.cs.cmu.edu/ide)
2. Copy the contents of `snakeSprint.py` into the editor
3. Press Run

## Features

- Arrow keys or WASD to move
- Poison bugs spawn randomly — eating one reverses your controls and slows the snake
- Eating normal food cures the poison effect
- Wall and self-collision end the game
- Press `R` to reset at any time

## What's next

A JavaScript port of this game is the first lesson on [myog.fun](https://myog.fun) — you'll rebuild it from scratch for the browser, then deploy it free with GitHub Pages.
